- ML Streamlit App

📂 **Project Folder:**  
[`MLStreamlitApp/`](MLStreamlitApp)

📄 **Project README:**  
[View detailed app description and run instructions](MLStreamlitApp/README.md)

**Description:**
An interactive web application designed to visually explore supervised machine learning models. Users can upload their own data or use built-in datasets, tweak model hyperparameters (like regularization or max tree depth) via sliders, and instantly observe how these choices impact model accuracy, precision, and error rates for both classification and regression tasks.

**Portfolio Integration:**
This project significantly elevates my portfolio by transitioning from static analysis scripts to fully interactive, user-facing tools. By allowing users to toggle hyperparameters and see real-time updates to performance metrics and charts (like ROC curves and Confusion Matrices), it demonstrates my ability to not only build accurate predictive models, but to effectively communicate and visualize their mechanics to non-technical stakeholders. It also highlights my proficiency in application deployment using Streamlit.

- Unsupervised ML App

📂 **Project Folder:**  
[`MLUnsupervisedApp/`](MLUnsupervisedApp)

📄 **Project README:**  
[View detailed app description and run instructions](MLUnsupervisedApp/README.md)

**Description:**
The Unsupervised ML App is an interactive Streamlit application that enables users to perform K-Means clustering and Principal Component Analysis (PCA) on tabular datasets. It allows users to upload their own data or use built-in sample datasets, and dynamically tune hyperparameters (like the number of clusters or principal components) to visualize the impact in real-time. This project builds on my understanding of machine learning by demonstrating how distance metrics and dimensionality reduction concepts are practically applied to identify hidden patterns and structures in unlabeled data.

**Portfolio Integration:**
This app complements my overall portfolio by taking machine learning beyond static code and into an interactive, deployable web format. By integrating user inputs, dynamic data visualizations, and live model training, it highlights my proficiency in interactive Python development. Furthermore, deploying the application on Streamlit Community Cloud demonstrates my ability to package data science projects into accessible, user-friendly applications.
